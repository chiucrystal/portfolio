Enabling self-serve and scalable program creation.

WiseTech Global (2025) · Team: 1 UX designer, 1 product manager, 4 developers Jump to solution
The design challenge

How might we build a self-serve program creation interface on a platform held together by manual processes and hardcoded configuration?

The tldr;

Problem:

Program setup took up to 3 weeks and required a developer to hard-code every configuration.

What I did:

I led design end-to-end as the sole designer, from research through handoff.

What changed:

The result was a self-serve creation interface that removed the dev dependency entirely.
